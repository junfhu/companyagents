## 软件概览

AI Delivery Operating System 是一套面向交付流程的工作流控制面，能够把一个需求从 intake 推进到 plan、review、execution、artifact 留痕和 runtime/supervisor 干预。

当前它已经可以支持：

- 任务创建与推进
- 计划与审核
- work item 执行跟踪
- artifact 管理
- runtime 自动推进
- supervisor 干预
- UI 中的全局和任务级控制

## 适合谁使用

- Workflow supervisor
- 项目经理或交付经理
- 审核角色
- 负责执行队列的团队负责人
- 做本地演示和产品验证的内部用户

## 当前限制

- 还没有完整认证系统
- 还没有生产级 worker fleet
- `Inbox` 和 `Templates` 页面还没实现
- 当前更适合本地开发、产品演示和设计验证
