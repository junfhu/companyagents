## Product Summary

AI Delivery Operating System is a delivery workflow control plane that moves a
request through intake, planning, review, execution, artifact tracking, and
runtime/supervisor intervention.

Today it already supports:

- task creation and progression
- planning and review
- work-item execution tracking
- artifact management
- runtime automation
- supervisor intervention
- global and task-level controls in the UI

## Intended Users

- Workflow supervisors
- Project or delivery managers
- Reviewers
- Team leads running execution queues
- Internal users doing local demos and product validation

## Current Limits

- No full authentication system yet
- No production-grade worker fleet
- `Inbox` and `Templates` pages are not implemented yet
- Best suited for local development, demos, and design validation
